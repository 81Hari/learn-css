a:visited
Link which is already visited and Using a concept called pseudo class (a:visited{})to change styling for visited links. In our case i set it to purple color.

a:hover, a:focus
Hovering is another state. So using pseudo class we can style that to. And a:focus makes the link to change color even you use tab to navigate a page

a:active
when you click and hold a link, it changes it's color
